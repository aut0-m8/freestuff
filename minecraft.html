PolyMC
Open source offline
Minecraft launcher
OpenJDK
Open source Java
Development Kit binary
Download OpenJDK first. Choose JDK 17 and above if you are
playing 1.18 and above. Choose JMC 8 if you are
playing 1.18 and below.
Extract the contents of the zip file to 

C:\Program Files\OpenJDK

Then open a Command Prompt window as Administrator and paste this in: 

setx JAVA_HOME "C:\Program Files\OpenJDK\jdk[version]"
setx PATH "%PATH%;%JAVA_HOME%\bin";

Press enter if anything remains in the terminal.
REMEMBER TO EDIT [version] WITH YOUR JAVA VERSION.

Download PolyMC. Expand the GitHub assets and download PolyMC-Windows-Setup.exe.
If Windows warns you, it is because the developer did not sign the exe.
After installation of PolyMC, choose your Java install directory.
Click Profiles > Manage Accounts
Click "Add offline"
Now exit the menu and click "Add Instance" and configure your instance.

You're done!
